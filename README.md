# Intervener Complexity in Natural Language Dependencies

**Course:** CGS410

## Overview

This project investigates the structural complexity of intervening nodes in dependency trees across twenty natural languages using Universal Dependencies (UD) Treebanks.

The study analyzes the syntactic properties of interveners and validates several linguistic hypotheses through statistical modeling and visualization.

---

## Objectives

- Analyze intervener arity across multiple languages
- Study subtree weight distribution
- Examine structural dependency relationships
- Perform Universal POS (UPOS) analysis
- Validate hypotheses using Poisson GLM and likelihood ratio testing

---

## Methodology

The project was implemented using Python and involved:

- Parsing Universal Dependency (.conllu) datasets
- Statistical analysis of dependency structures
- Cross-linguistic comparison across 20 languages
- Data visualization
- Hypothesis testing

---

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Statsmodels
- CONLLU

---

## Dataset

Universal Dependencies (UD) Treebanks

---

## Key Findings

- Interveners generally have low arity.
- Most interveners consist of very small subtrees.
- Structural independence dominates dependency relations.
- Nouns and Adpositions are the most common intervener categories.
- Statistical analysis supports the proposed linguistic hypotheses across multiple languages.
